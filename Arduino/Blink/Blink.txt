/*
	Blink
	LEDを一秒間点灯、消灯を繰り返す。
	このサンプルコードはパブリックドメインです。
*/

// 13番ピンは大抵のarduinoボードでLEDにつながっている。
// これに名前をつける。
int led = 13;

// "setup"ルーティンはボードの電源を入れたときやリセットを押した時に一度だけ行われる。
void setup(){
	//　そのデジタルピンを出力に設定する。
	pinMode(led,OUTPUT);
}

// "loop"ルーティンは繰り返し行われる。
void loop(){
	digitalWrite(led, HIGH);	//　LEDを点灯
	delay(1000);			//　一秒待つ
	digitalWrite(led, LOW);		//　電圧を落としてLEDを消灯
	delay(1000);			//　一秒待つ
}


"//"ではじめるか"/*""*/"で囲まれた部分はコメント行。arduinoには無視される。
「真のハッカーはコメントを書く」そうです。
	int led = 13;
"led"という名の整数型の変数を作り、それに13を代入する。
";"は行の終わりを示す。忘れずに入れること！！入れ忘れたり":"と間違えるエラーはとても多いです。
int led=13;　のようにスペースを入れなくてもよい(スペースは無視される)が、入れたほうが見やすい。
	void setup(){
初期設定のために必ず必要な一行。
"{"の後の行は段下げが行われる。
	pinMode(led, OUTPUT);
変数"led"(13)を出力に。このように書くことでarduinoに"led"がピンの番号であると指示する。
大文字小文字は区別されるので間違えないように。
	}
"{"と"}"は必ずワンセットで使われる

	void loop(){
以下実行される内容
	digitalWrite(led, HIGH);
"led"(13番ピン)を"HIGH"に。unoの場合は５V出力。
	delay(1000);
delay(ms)の単位はミリセカンド。
	digitalWrite(led, LOW);
"led"を０Vに。
	delay(1000);
	}


スケッチを書き終えたら左上のチェックを押してください。
間違いがあれば下にエラーが表示されます。

"ツール"  "マイコンボード" でArduino Uno を選択します。
"ツール"  "シリアルポート" で正しいCOMポートを選択します。
Windowsの場合はデバイスマネージャを開いて"ポート(COMとLPT)"でarduinoが認識されているポートを確認してください。
上左から2番目の矢印アイコンを押して転送です。
arduino上のLEDが短く点滅した後、一秒ごとに点滅すれば成功です。
