# arduino_basic_ws
ワークショップ「Arduino入門」のパッケージ。

#目標
ワークショップをパッケージ化していつでも開催できるようにする。

# WS概要
詳細は[01_Introduction.md](配布資料/01_Introduction.md)を参照。

## 目安時間
- 2時間

## 参加条件
- 小学校高学年以上推奨
- Arduino開発環境を自分で用意できる

##目安参加人数
- 同時に３人/１インストラクタ

# 構成
- Arduino：WSで使用するArduino用のスケッチプログラム
- 配布資料：当日使用する資料
- 事前準備：開催準備に関係する資料
